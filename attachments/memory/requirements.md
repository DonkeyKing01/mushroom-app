# 菌类探索平台 (Fungal Discovery Platform)

## 产品语言
中文

## 要构建什么
一个融合科技美学与菌类科普的沉浸式 Web 平台。核心体验围绕 WebGL 流体动力学视觉呈现，通过数据雕塑、极光色谱和粒子系统构建未来主义的菌类知识探索空间。用户可以通过扫码识别菌类、浏览物种档案、查看实时地图分布、在交互实验室中探索菌丝生长模拟，以及获取食用菌菜谱和安全预警信息。

## 目标受众
跨界创意工作者——设计师、艺术家、数字创意从业者。他们对前沿视觉技术和自然美学的交叉领域充满好奇，追求独特的视觉体验和灵感来源。核心用户场景包括：寻找设计灵感、探索自然与数字艺术的融合、获取高质量的菌类视觉素材参考，以及在社区中分享创意作品。

## 设计氛围与感受
极致的科技未来主义（Tech-Futurism）与有机生命力的矛盾融合。视觉上呈现高对比度的暗黑美学：纯黑背景上流动着极光色谱（青色、深紫、洋红、金黄）的自发光流体，模拟菌丝网络的生长与扩散。动态效果具有明显的惯性阻尼，如海洋波浪般的非线性节奏。交互响应产生磁场般的扰动，粒子受力偏离轨道并产生色彩偏移。整体氛围应让用户感受到置身于微观生物世界的数字显微镜下，既神秘又充满探索欲。

## 功能与页面
本版包含五大核心页面及其导航体系：

**1. 发现首页 (Discovery Home)** - 叙事性入口，引导用户进入菌类世界
- Hero Section：**完整 Three.js/React Three Fiber WebGL 场景** - 2000 个粒子组成的菌丝网络实时模拟，基于卷曲噪声的流体动力学驱动，鼠标交互产生磁场般的扰动效果，粒子受力偏离轨道并产生弹性回归，采用极光色谱（青色、洋红、紫色、金色）自发光材质与 AdditiveBlending 混合模式，整体场景缓慢旋转营造有机生命感。左侧内容区使用渐变背景和毛玻璃效果确保文字可读性。
- Feature Grid：五大功能磁贴布局（档案馆入口展示热门物种轮播、实时地图缩略动态图层、实验室入口显示在线人数等）
- Curated Content：左侧展示"每日菌星"随机物种卡片，右侧显示科普动态和季节性高风险菌预警流

**2. 档案馆 (Archive)** - 核心知识引擎，采用"侧边筛选+右侧结果"的经典高效布局，已完整接入 SuperunCloud 数据库
- 顶部：面包屑导航 + 语义搜索框（实时搜索物种名、拉丁学名）
- 中间主界面：
  * 左侧：可折叠的多维筛选器（单选框：食用安全性；复选框：形态特征）
  * 右侧：物种卡片网格（每张卡片包含真实图片、变色标签、悬停 3D 提示）
- 底部：分页控件 + 法律免责声明
- 数据库架构：7张规范化表（categories, species, species_images, similar_species, anatomy_data, ecology_data, cooking_notes）
- 物种详情页：剥洋葱式信息披露，完整关联查询
  * ID Card：多角度高清画廊 + 核心特征摘要（科学名、食用等级、季节性）+ 相似种对比（易混淆的有毒品种警示）
  * Interactive Anatomy：3D 解剖模型占位 + 科学参数表（KOH 试剂反应、孢子印颜色、菌环类型）
  * Ecology & Narrative：生态网络关系图（共生树种）+ 烹饪与中毒预警（红字声明）

**3. 地图社区 (Map Community)** - 实时地理分布与用户分享（首版框架+占位内容）

**4. 交互实验室 (Interactive Lab)** - 菌丝生长模拟和数据可视化实验（首版框架+占位内容）

**5. 菜谱 (Recipes)** - 交互式 AI 菜谱生成系统（前端可视化完整实现）
- 顶部：毛玻璃搜索框 + 可食用菌类横向滚动卡片库
- 中部：WebGL 热气特效背景 + 数字汤锅可视化（俯视视角，带脉冲圆环和渐变汤汁效果）
- 左侧：已选食材列表（最多5种，支持移除）
- 交互：拖拽食材卡片至汤锅，触发涟漪动效和 toast 提示，点击"生成创意菜谱"按钮跳转子页
- AI 菜谱详情页：左侧 AI 厨神笔记（菜名、难度、时长、份量、食材清单），右侧分步骤滑块（步骤进度条、当前步骤详情、温度/时长标注、完成标记），底部生存指南（关键安全警告红色高亮显示、安全等级进度条）

所有页面遵循极简 UI 原则：无衬线/等宽字体，元素放置在画面极端角落（边缘对齐），确保不遮挡中心区域视觉主体。导航采用悬浮式磨砂玻璃效果，融入整体暗黑基调。

## 技术决策
- 目标平台：桌面端优先（1440px）
- 视觉实现：首版采用静态渐变背景 + CSS 微动效，后续可升级为完整 Three.js WebGL 效果
- 首版范围：完整五页框架，首页完整功能，其他页面使用占位内容
